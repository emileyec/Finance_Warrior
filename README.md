# Finance Warrior

Finance Warrior is a Java application designed to help users manage their financial goals, track progress, and visualize important financial data.  
This project was built using **Java**, **Maven**, and **NetBeans**.

---

##  Features

- Add, edit, and track financial goals (quests)
- Set priority, start date, and end date for each goal
- Dashboard for viewing goals and progress
- Persistent data storage
- Simple, intuitive UI

## Building the Project
1. Download the Project folder.
2. Open a terminal/command prompt.
3. Navigate to the project directory. EX:
```bash
cd path/to/Finance_Warrior
```
4. Build the project using Maven:
```bash
mvn clean package
```
5. After building, the runnable JAR will appear in:
target/Finance_Warrior-1.0-SNAPSHOT.jar
6. Run the application using:
```bash
java -jar target/Finance_Warrior-1.0-SNAPSHOT.jar
```
